| 단어명 | 설명 |
|--------|--------|
| **FastAPI** | Python으로 API를 만들 수 있는 프레임워크. 빠르고 간편하게 API를 만들 수 있다.|
| **pyproject.toml** |  Python프로젝트의 정보를 선언하는 언어, JAVA의 build.gradle과 비슷 |
| **Gunicorn** | WSGI서버(Python 웹 어플리케이션을 실행하기위한 서버) |
| **HTTP Message** | 서바와 클라이언트가 통신을 위해 주고받는 메시지  |
| **HTTP Request Method** | 웹서버에게 작업을 요청하는 명령 메소드|
| **HTTP Status Code** | 현재 처리 상태를 나타내는 코드 |
| **HTTP URL** | http에게 요청하는 주소 |
| **Query string** | URL에서 ?key=value 형태로 데이터를 전달하는 방법 |
| **Path Variable** | URL 경로 자체에 포함된 변수 값 |
| **JSON** | 구조화된 데이터를 표현하기 위한 데이터 포맷 |
| **REST API** | 정보를 교환하는 방식의 아키텍처 스타일 |
| **디자인패턴** | 자주 등장하는 소프트웨어 문제를 해결하기 위한 설계 템플릿 |
| **Database** | 데이터를 모아둔 저장소 |
| **SQL** | Database를 사용할때 사용하는 언어 |
| **데이터 정의어** | DDL - 데이터 구조를 정의하는 언어 (테이블생성, 테이블수정, 테이블삭제)|
| **데이터 조작어** | DML - 데이터를 조작하는 언어 (데이터 조회, 데이터 삽입, 데이터 수정, 데이터 삭제)|
| **데이터 제어어** | DCL - 데이터를 제어하기 위한 언어 (권한 설정) |
| **JOIN** | 테이블을 합쳐(조인해서) 원하는 데이터를 조회 |
| **정규화** | 테이블을 분해해서 데이터의 이상현상을 제거하기 위한 과정 |
| **암호화** | 데이터를 다른문자로 변환하여 원본데이터를 볼 수 없도록 가리고, 복구할 수 있는 기능|
| **bcrypt** | 주로 비밀번호에 사용하는 단방향 해싱함수 |
| **stateful** | 서버가 클라이언트의 이전 요청 정보를 기억 |
| **stateless** | 서버가 클라이언트의 이전 요청 정보를 기억하지 못함 |
| **connectionless** | 비연결형 데이터 전송방식 (UDP) |
| **인증·인가** | 사용자의 신원을 확인하는 보안 절차, 인증으로 식별하고, 인가로 접근 권한을 줌 |
| **쿠키** | 클라이언트의 브라우저에 저장해두는 데이터 |
| **세션** | 서버에 저장해두는 사용자의 상태 정보|
| **프레임워크** | 개발에 필요한 기능을 구현해둔 프레임 |
| **라이브러리** | 개발에 필요한 기능을 호출해 사용할 수 있는 코드의 집합? |
| **API** | 애플리케이션들이 데이터를 주고받는 명세서 |
| **페이징** | 데이터를 나누어서 페이지 단위로 처리하는 방법 |
| **HTTPS** | HTTP의 전송 계층에 암호화를 적용해 보안을 강화시킨 통신규약 |
| **log** | 컴퓨터의 기록을 남겨둔 파일 |
| **HTML** | 웹페이지의 구조를 만들고 영역을 구분하는 언어 |
| **CSS** | 웹의 스타일을 저장해둔 스타일시트 |
| **Boxmodel** | HTML의 구조를 박스형식으로 나누어 놓은 구조 |
| **Flex** | 웹페이지의 요소를 유연하게 배치하기 위한 레이아웃 모델 |
| **Grid** | 웹페이지의 요소를 격자형태로 구성하여 배치하기 위한 레이아웃 모델|
| **DOM** | HTML 문서를 객체로 표현하여 JavaScript로 조작할 수 있게 만든 객체 모델 |
| **CSSOM** | CSS 스타일 정보를 객체 구조로 표현한 모델 |
| **Event** | 사용자가 웹페이지에서 하는 동작 |
| **Fetch** | 브라우저에서 HTTP 요청을 보내고 응답을 받는 비동기 API |
| **Lottie** | JSON기반의 애니메이션 파일 |
| **디버깅 (Debugging)** | 버그를 찾아서 수정하는 과정 |
| **Search Engin Optimization** | 검색 엔진에 웹사이트를 최적화하여 검색 결과 상위를 노리는 기술 |
| **Search Engine** | 웹 정보를 수집·색인하고, 사용자의 검색 질의에 맞는 결과를 제공하는 시스템 |
